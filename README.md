## Start

Use `make create` to create Firefox Container and add to Host (Desktop, CLI, and Menu).

## Launch

### CLI

`lxff`

### Desktop

Use LXFF on Desktop.

### Menu

Type "LXFF"

## Help

Use `make help` to see all commands.
